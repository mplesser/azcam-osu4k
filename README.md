# osu4k

## Purpose

This repository contains *azcam* files for the OSU4k CCD camera.

## Installation Examples and Notes

* `python -m venv env`
* `.\venv\Scripts\activate.bat`
* `pip install -r requirements.txt`

Execute the following scripts:

* ``start_azcamconsole.py``
